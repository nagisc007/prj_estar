## Characters

- ロムス……傭兵。愛する女を殺した敵の魔物を探して旅を続けている。
- ミリア……ロムスの幼馴染。魔物により殺された。
- ミラ……ロムスを介抱してくれる村の女性。口が利けない。

## Stages

- 村……山の麓にある村。魔物の人が拐われて困っている。
- メギド山脈……大陸でも最も険しいと云われる山脈。
- 洞窟……メギド山脈の中にある横穴。溶岩質で沢山ある。
